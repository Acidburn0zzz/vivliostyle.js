# Contribution Guide

> To get started, <a href="https://www.clahub.com/agreements/vivliostyle/vivliostyle">sign the Contributor License Agreement</a>.

## Setup

```
git clone https://github.com/vivliostyle/vivliostyle.git
cd vivliostyle
yarn install
yarn bootstrap
```

## Build

```
yarn build
```

## Watch

```
yarn dev
```

## Lint & Format

```
yarn lint
yarn format
```
