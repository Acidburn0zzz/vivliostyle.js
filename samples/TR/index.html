Vivliostyle TR demo

This is a semi-random selection of W3C TR documents, and a few editors drafts and IPDF specs as well. Each document can be viewed via two links. The first one (“Classic”) shows it in its usual form. The second one (“Vivliostyle”) shows the same document in a paginated yet responsive view using Vivliostyle.js and a custom user stylesheet

The default setting is to show a single responsive page, but by clicking on the icon, you can set a predefined page size (for example A4), or switch to a spread view, to see two pages side by side.

This is a work in progress: the stylesheet needs more work to deal with all the variations of W3C documents, and Vivliostyle.js isn't free of bugs either. On top of that, there are a few more features we would like to include in Vivliostyle.js to improve this even further. Even then, this should be interesting.
