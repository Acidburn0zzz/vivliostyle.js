Vivliostyle TR demo

This is a semi-random selection of W3C TR documents, and a few editors drafts and IPDF specs as well. Each document can be viewed via two links. The first one (“Classic”) shows it in its usual form. The second one (“Vivliostyle”) shows the same document in a paginated yet responsive view (spread view if window is wide) using Vivliostyle.js and a custom user stylesheet
